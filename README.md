# Skateboard-Phyisics-Game
A simple phisics-based game using Phaser, where the player traverses with a skateboard. 
---------------
The game includes three gameplay scenes where the first two walk the player through basic controls of riding the skateboard. After the player is finally comfortable, they can explore a final scene with more obstacles and more fun!

The git commit history will start empty and show detailed commit messages for each update

The code will collect keyboard input as well as mouse input to move the player to their liking

The code will use the Arcade physics engine with collisions and/or forces

The project will contain 3+ distinct core gameplay scenes with a title scene and loading screens in between each playable scene
